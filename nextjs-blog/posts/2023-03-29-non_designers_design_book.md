# ノンデザイナーズ・デザインブック

## ４つの基本原則
- コントラスト
  - 違うならはっきり違いを出すべき
  - 書体、色、サイズ、太さ、形、空きなど
- 反復
  - 同じ要素は同じ見た目にしとけってこと
- 整列
  - 揃えろってこと
- 近接
  - 似たやつはまとめてグループ化すると認識が楽

## 近接の原則 - Proxymity
- 要素と要素の間隔や余白に目をつける
- まず最初に適用する原則
- 目線の迷いを減らそう
- 空いてるから埋めるはやめろ

## 整列の原則 - Alignment
- 要素が離れていても、整列していれば繋がりが見えてくる
- 中央揃えはラインが曖昧になる
  - フォーマルな場やおとなしい印象
- 左右揃えはラインがはっきり出るので強さが出る
- 基本は揃えを１種類で使う
- インデントは1em(約スペース２個)
- 最初の段落にインデントは不要（英語文化かも）
- すべての要素が他とつながっているように配置する
- 画面の一体化と組織化が目的
- 強い整列を見つけ出し、維持する
- ルールを破るにはルールを知る必要がある
- 離れていても揃えることができる場合がある
- 中央揃えにするクセはやめろ　意識的に選択しろ

## 反復の原則 - Repetition
- 複数ページにまたがって使われるノンブル、罫線、デザイン
- 同一ページでも利用可能
- 一貫性ともいう
- 目的は一体化と視覚的な面白さを加えること
- 反復は作り出せるし、反復のために要素を追加することもできる
- コントラストの価値を意識した量に抑える(次章)

## コントラストの原則 - Contrast
- 一番面白いものや一番重要な部分を目立たせる
- 違うものが似てしまうことを避ける　徹底的に違いを出す
- 目的はおもしろみを作り出すことと、組織化を支援すること
- 書体、太さ、色、形、サイズ、空きなど

## コントラストの章にあった手順
- Times New RomanやArielやHelveticaを捨てます。
- Sandがあればそれも捨てます。
- **中央揃え**を捨てます。後で試せるのでまず捨てます。
- 一番面白いもの、重要なものを**強調**し、読者を集中させます。
- 枠でなく**空き**を使って情報をまとめます。
- **反復**できる要素を見つけます。
- **コントラスト**を加えます。

## カラーを使ったデザイン - Design with Colors
- カラーホイール(12色)
  - 赤青黄が原色　隣通しを混ぜていく
  - 緑紫オレンジが２次色
  - 残りが３次色
- 補色
  - 真向かいにある色　アクセントになる
- トライアド
  - 等しい距離にある３つの色（赤青黄みたいなの）
  - うまく調和して心地よくなる　らしい
- スプリット・コンプリメント（・トライアド）
	- 真向かいの補色の両隣を使うと、少し洗練された鋭さをもつ感じらしい
- 類似色
  - 隣り合う２色または３色のこと　これも調和らしい
- ヒュー
  - そのままの色彩（あるいは一番鮮やかな純色）
- シェード
  - 黒を加える
- チント
  - 白を加える
- 単色的な組み合わせ
  - 一つのヒューに対応する複数のチントとシェードで組み合わせる
  - グレースケールは黒の単色的な組み合わせ
- トーン
  - ヒューンの濃淡の特定の質？？
  - トーンが近いと混ざって見づらい
- 暖色　全面に出て見える
- 寒色　背景に引っ込んで見える
- CMYKとRGB
  - 印刷向きと画面上での表示向き
  - 相互の変換には劣化が生じる
  - RGBで作って最後にCMYK変換するのが主流らしい

## 活字(と人生) - Type(& Life)
- 協調関係
  - 一種類の書体ファミリーだけを使った場合に生じる
  - 調和を保てるが退屈になりがち
- 衝突関係
  - 似ている書体を組み合わせた場合に生じる
  - 中途半端に近いせいで混乱を招く
- コントラスト関係
  - 互いにはっきり異なる書体を組み合わせた場合に生じる
  - 視覚的に魅力的でエキサイティング
- 活字でコントラストを付ける６の方法
  - サイズ
  - 太さ
  - 構造
  - フォーム
  - 方向
  - 色
- ほとんどが組み合わせになり、単独で効果を発揮することはあまりない
- 組み合わせがいまいちなときは、違うところではなく同じところを見る
  - 問題を引き起こすのは*類似性*

## 活字のカテゴリー - Categories of Type
- 大体６のカテゴリーにおさまる
  - Oldstyle
  - Modern
  - slab serif
  - Sans serif
  - script
  - Decorative

### オールドスタイル - Oldstyle
- 尖ったペンで書いた感じ
- 必ずセリフがある
- 小文字のセリフは斜めになっている
- セリフと本体はブラケティングと呼ばれるカーブで接続している
- ペン字がベースなので、曲線部分(ストローク)には細いところと太いところがある
  - thick / thin transition
- ストロークの細いところに線を入れると斜めになる（ストレスと呼ぶ）

目立たないグループで、長文に向いている
ほぼみんないっしょに見える

### モダン - Modern
- ペンからメカニカルへ
- セリフはあるが傾斜がなく、水平　かつ細い
- ストロークの差が激しく、コントラストがある
- ストレスが垂直
- 冷たくエレガントな傾向

長文に向かないグループ
大きいサイズで使うと目に付きやすい
小さくすると太い線だけ見えるようになる（dazzling)

### スラブセリフ - Slab serif
- モダン書体が更に太くなった
- ストロークの差があんまりない
- Clarendonと呼ばれることもある（同名のフォント）
- Egyptianと呼ばれることもある（西洋文明のエジプトブーム時期に流行ったから）
  - エジプトっぽい名前が多いのはそのせい
- モダンベースだからストレスは垂直
- 小文字のセリフは水平で太い
- slabは厚板を意味する

可読性に優れるため長文に利用可能
画面が黒くなる傾向あり
子供向けでよく使われる

### サンセリフ - Sans serif
- sansはフランス語で*無い*を意味する
- ２０世紀に入るまで流行らなかったらしい
- ストロークの差はなく、太さが一定（monoweight)
- 太さが変わらないのでストレスもない

かの悪名高いHelvetica、Arial、Verdanaはこのグループ
もしこれらしかない場合、投資しろとのこと
ごく少数、太さの推移があるサンセリフ書体があるが、セリフ書体ともサンセリフ書体とも類似するため組み合わせると衝突を起こす
慎重に扱うこと

### スクリプト - script
- 手書き文字っぽいやつ
- 控えめに使わないとくどい
- 絶対にせんぶ大文字にしてはいけない
- 長文にも使うな
- でも大きいサイズで使うとインパクトがある

### デコラティブ - Decorative
- 非常に目立つが使い所が限られる
- 安いらしい

### まとめ

  - xハイトの底面より下って書いたほうがいいのでは

## 活字のコントラスト - Type Contrasts
- size
  - はっきり違うサイズにしろ　12ポイントと65ポイントとか
- weight
  - はっきり違う太さにしろ　フォントによっては太字が細すぎる
- structure
  - 組み合わせるならサンセリフとセリフが無難　もちろん大きさとかも変えろ
- form
  - 外形のこと　Gとgの違いのこと　全部大文字だとシルエットが長方形になって読みづらい
  - イタリック体はe f a y が専用になってたりする　スクリプトとは衝突しがち
  - ローマン体はまっすぐ文字がたってるやつのこと　組み合わせやすい
- direction
  - 方向　斜めにするなら理由をはっきりしろ
  - 横長フォントを水平に配置したり、縦長フォントを縦長の段組にいれたりしろ
- color
  - 暖色は目に付きやすく、寒色は多めに使わないとバランスが取れない
  - 白地に黒文字一色でも、太くしたり大きくしたりで色の印象が出る

### まとめ
- コントラストを組み合わせよ
  - 大文字ローマン体と小文字のイタリック体など
  - 方向、サイズ、太さなどなど
- 「関係の力学を言葉に出来れば、それを支配することができる」
  - 良し悪しを理論的に説明できるようになれということ
- 「問題の名前を言えて初めて、解決策を生み出すことができる」
- size　臆病にならないこと（めっちゃ小さくしたりでかくしろ）
- weight　太字と細字でコントラストを出せ
- structure　異なるカテゴリーのフォントを組み合わせろ
- form　フォームの異なるフォントを組み合わせろ　ローマンvsイタリックとか
- direction　角度だけでなく、水平と段組の対比まで広げろ
- color　暖色と寒色。黒いテキストの「色」。
